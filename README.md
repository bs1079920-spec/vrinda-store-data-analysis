## What I Did
- **Cleaned** 31,000+ raw order records — fixed missing ship-state values, standardized channel names, removed cancelled-only noise
- **Analyzed** customer demographics, channel performance, and regional sales using PivotTables
- **Built** an interactive, slicer-driven dashboard for live exploration

## Findings

| Insight | Recommendation |
|---|---|
| Women (30–49) drive 64% of sales | Prioritize this segment in targeting & product testing |
| Amazon, Myntra, Flipkart = 80% of orders | Concentrate ad spend on top 3 channels |
| Maharashtra, Karnataka, UP lead all states | Pilot region-specific campaigns here first |
| Sales fell 28% from March peak to December | Run a Q3 promotional push to offset the dip |
| 92% delivery success rate | Confirms the dip is a demand issue, not fulfillment |
